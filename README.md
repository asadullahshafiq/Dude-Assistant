## 📥 GitHub Se APK Kaise Hasil Karein?

### Step 1 – GitHub Account Banayein
1. **https://github.com** kholo (PC par)
2. **Sign Up** karo → free account banao
3. Email verify karo

### Step 2 – Naya Repository Banao
1. GitHub pe "+" icon → **New repository**
2. Name: `dude-assistant`
3. Public ✓
4. **Create repository** click karo

### Step 3 – Files Upload Karo
1. **"uploading an existing file"** link click karo
2. Is ZIP ki saari files drag & drop karo
   (structure maintain karein – folders bhi)
3. **Commit changes** click karo

### Step 4 – APK Build Hogi Automatically! ✨
1. **Actions** tab pe jao (upar)
2. `Build APK` workflow dikhe ga
3. Kuch minute wait karo (5-10 min)
4. Green ✓ aane pe click karo
5. **Artifacts** section mein `DudeAssistant-APK` download karo
6. ZIP extract karo → `app-debug.apk` milegi

---

## 📲 APK Install Kaise Karein (Samsung A10s)?

1. **Settings** → **Biometric and Security** → **Install Unknown Apps**
2. Browser allow karo
3. APK file tap karo → **Install**

---

## 🛠️ First Time Setup (App Open Karne Par):

1. **Microphone** → Enable dabao
2. **Overlay** → Enable dabao (permission page pe Allow)
3. **Accessibility** → Settings khulenge → Dude Assistant dhundh kar ON karo
4. **Default Assistant** → Settings → Apps → Default Apps → Digital Assistant → Dude
